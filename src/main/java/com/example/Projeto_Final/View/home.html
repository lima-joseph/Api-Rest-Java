Projeto de API com Spring - Gestão de Clientes e Produtos

No que consiste?
Este projeto consiste em uma API desenvolvida em Java utilizando o framework Spring Boot, que permite o gerenciamento de clientes e produtos. A API oferece funcionalidades completas de CRUD (Create, Read, Update, Delete), além de várias opções de busca para facilitar o acesso aos dados de clientes e produtos.

Funcionalidades

  • Cliente
    Cadastrar Cliente, Buscar Todos os Clientes, Buscar Cliente por Código, Buscar Clientes por Parte Inicial do Nome, Buscar Clientes pelo E-mail, Buscar Clientes por Combinação de Nome e E-mail Remover Cliente (Pelo registro completo ou Pelo código), Atualizar Cliente
  • Produtos
    Cadastrar Produto, Buscar Todos os Produtos, Buscar Produto por Código, Buscar Produtos por Parte Inicial da Descrição, Buscar Produtos por Preço Inferior a um Valor, Buscar Produtos por Marca, Buscar Produtos por Combinação de Marca e Preço, Remover Produto (Pelo registro completo ou Pelo código), Atualizar Produto
